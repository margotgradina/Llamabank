Add new accountholder to Bankaccount


To add a new accountholder to this bankaccount, fill in the id of the client you wish to add.
The id can be found on the userdashboard of the client, after logging in.
Write down a 5 digit security code and communicate the code and bankaccountnumber (in case of IBAN, this is the part after 'LAMA')
with the new Accountholder in a safe way.

After clicking "Request new accountholder", the accountmanager reviews your request to add the accountholder.
The new accountholder can add the bankaccount via the userdashboard > Open/add new BankAccount > Add Existing Account by filling in the bankaccountnumber and the 5 digit security code.

This can only be done once the request has been approved by the accountmanager.
